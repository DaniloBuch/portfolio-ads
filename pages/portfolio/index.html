Meus Projetos

Aqui estão alguns projetos onde aplico minhas habilidades para resolver problemas práticos, transformando dados e lógica de negócio em aplicações funcionais.

Meteora

O Meteora é um e-commerce fictício de moda em constante desenvolvimento, nesta página foi implementado o sistema de 'carrinho de compras'. Este projeto visa aprimorar as habilidades em React, e apresentar a Context API como uma solução para gerenciamento de estados globais de uma aplicação React.

Veja você mesmo! GitHub

Freelando

O "Freelando" é um projeto prático que simula a criação de uma aplicação web para o mercado de trabalho, nessa página simula o cadastro de um freelancer. O objetivo principal é consolidar as habilidades em React, abordando desde a componentização e navegação com React Router, até o gerenciamento de estado com a Context API.

Veja você mesmo! GitHub

Meteora

O "Organo" é um projeto prático que simula a criação de uma plataforma para organizar equipes de trabalho. Nesta página, é possível cadastrar novos colaboradores e distribuí-los em times. O objetivo principal é consolidar as habilidades em React, abordando desde a criação de componentes (como formulários, cards de colaborador e seções de times) até o gerenciamento de estado para manipular a lista de participantes de forma dinâmica.
